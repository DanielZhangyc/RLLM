# RLLM

一个由LLM驱动的RSS阅读器

## Features

- ✅ 基础RSS阅读
- ✅ 文章/语段阅读与收藏
- ✅ AI文章摘要生成
- ✅ AI文章洞察分析
- ✅ 每日阅读AI总结
